Multiview plugin for jQuery

Sample Use:
http://facherisdesign.com/yacts.php

*Description
Multiview is a gallery plugin for jquery that uses divs instead of images for slides for more control of content.

*Use

Set up HTML like this:

<div class='gallery'>
    <img class='thumb' src='images/thumb.jpg' /><div class='slide'>Slide1 content here.</div> 
    <img class='thumb' src='images/thumb.jpg' /><div class='slide'>Slide2 content here.</div> 
    <img class='thumb' src='images/thumb.jpg' /><div class='slide'>Slide3 content here.</div> 
    <div class='navigation'></div>
</div> 

Be sure jQuery is imported.

Call the plugin using:
$('.gallery').multiview();
